Permissions for Shop Owners:

  1. Admin creates a shop

  2. Admin adds member id of Shop owner into field shopOwnerID

  3. System assigns local role "Shop Owner" to shop content object to member
  with entered id

  4. System assigns local role "Manager" to subfolder of shop ot member with 
  entered id

  => Shop owner can edit shop info but can't delete shop
  => Shop owner can add products, categories, but can't delete subfolders
